## Task 3 — UI Data Requirement Map

Create a complete mapping from UI sections to required data fields.

Use this format:

```text
UI Section:
Required fields:
Current source:
Future source:
Notes:
```

Classify future source as one of these:

* `DATASET_DIRECT`

  * Comes directly from processed CSV.
  * Example: player name, age, team, league, position, goals, assists, minutes.

* `MODEL_OUTPUT`

  * Comes from trained model prediction.
  * Example: predicted market value.

* `DERIVED_SCORE`

  * Calculated from multiple stats.
  * Example: AI Quality Index, Attack Score, Playmaking Score, Dribbling Score.

* `TEMPLATE_GENERATED`

  * Generated by rule-based text templates.
  * Example: strong sides, weak sides, development focus, AI comment.

* `FRONTEND_CONFIG`

  * Pure UI configuration.
  * Example: shirt color, badge placeholder, club color, sidebar grouping.

* `MISSING_OR_UNKNOWN`

  * Needed by UI but no clear source found yet.

## Recovered Fields

Only the five fields approved by the audit were added:

| SofaScore column | Generated raw metric destination |
| --- | --- |
| `appearances` | `rawMetrics -> Overview -> Appearances` |
| `matchesStarted` | `rawMetrics -> Overview -> Matches Started` |
| `penaltyGoals` | `rawMetrics -> Attack & Shooting -> Penalty Goals` |
| `totalPasses` | `rawMetrics -> Playmaking & Passing -> Total Passes` |
| `totalContest` | `rawMetrics -> Physicality & Duels -> Total Contest` |

If a selected player lacks a source value for one of these fields, the exporter still falls back to the configured `UNKNOWN` value for that specific metric.
